## Typing Special Letters with US Keyboard

For MS Office:

    ß: Alt + 0223 (esszett)
    ä: Alt + 0228 prounounced as ae
    ö: Alt + 0246
    ü: Alt + 0252
    Ä: Alt + 0196
    Ö: Alt + 0214
    Ü: Alt + 0220

For Markdown:

    a-umlaut - &auml;
    o-umlaut - &ouml;
    u-umlaut - &uuml;
